🚀 Siretizator API

Bienvenue ! Cette API vous permet de retrouver les numéros SIRET à partir d'informations partielles sur des agents économiques.

🔧 Endpoints disponibles

  • POST /siretize : recherche d’un seul agent
  • POST /siretize_bulk : recherche pour plusieurs agents
